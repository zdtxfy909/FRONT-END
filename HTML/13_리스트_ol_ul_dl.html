리스트 작성시 사용되는 태그(ol, ul)

ol, ul 자녀태그에는 li 태그만 사용가능

순서가 있는 리스트 : ol - ordered list

  1. 사과
  2. 배
  3. 복숭아
  4. 수박

순서가 없는 리스트 : ul - unordered list

  • 사과
  • 배
  • 복숭아
  • 수박

계층구조 목록 작성 : ul + ol

  • 사과
    1. 최상품사과
    2. 상품사과
    3. 일반사과
    4. 불량사과
      • 벌레먹은 사과
      • 썩은 사과
      • 덜 익은 사과
  • 배
  • 복숭아
  • 수박

정의목록 작성(dl, dt, dd)

HTML
하이퍼 텍스트 마크업 언어(영어: Hyper Text Markup Language, HTML, 문화어: 초본문표식달기언어, 하이퍼본문표식달기언어)는 웹 페이지 표시를 위해 개발된 지배적인 마크업 언어다.
CSS
CSS(Cascading Style Sheets)는 HTML 또는 XML과 같은 마크업 언어(SVG, MathML 또는 XHTML과 같은 XML 방언 포함)로 작성된 문서의 프레젠테이션을 설명하는 데 사용되는 스타일 시트 언어입니다.
